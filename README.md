# Email Validator without Regex

## EMAIL Requirements

#### Completed
1) only one '@', should not be first
2) no spaces 
3) '.' should not be last or second last
4) min length of email = 6 (ex- g@g.in), lavi@gmail.com 
5) index 0 sgould be an alphabet 
6) email can have uppercase
